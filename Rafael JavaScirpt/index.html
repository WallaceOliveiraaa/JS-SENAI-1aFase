Atalhos do VS Code

Atalhos gerais:

Vamos ver alguns atalhos legais do VS Code que vão deixar a nossa vida mais alegre, supimpa e feliz.

alt+z: Quebra automática a linha

alt + setas p/ cima: Move a seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor está.

shift + alt + setas p/ cima: Move uma cópia da seleção para cima/baixo no arquivo. Se nada estiver selecionado, movimenta a linha em que o cursor está.

ctrl + espaço: chama o autocompletar de um contexto, como selecionar uma imagem ou um texto.

ctrl + ; (ponto e vírgula) : comentário/descomentário

shift + ! : "html"
